A brief history of StormTrack

This project is a work in progress and was created using Quarkus by two web developer students, Vinicius and Rodrigo. Our goal is to provide a free, open-source and online-based meteorological service for everyone, that contains all you need without being complicated and messy to use.

You can see our source code in our GitHub repository

This website utilizes OpenWeatherMapAPI, a global weather data system and UnsplashAPI, an image API.
